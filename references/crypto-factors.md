# 虚拟货币胜率评估因子

## 因子总览

共12个核心因子 + 3个宏观调整因子：

| 维度 | 因子数量 | 权重合计 |
|------|----------|----------|
| 估值 | 3个 | 32% |
| 趋势 | 3个 | 23% |
| 资金 | 3个 | 22% |
| 情绪 | 3个 | 23% |
| 宏观调整 | 3个 | ±10% |

---

## 一、估值维度（32%）

### 1.1 AHR999囤币指标（12%）

**创建者**: 微博用户 ahr999（九神）

**公式**: 
```
AHR999 = (BTC价格 / 200日定投成本) × (BTC价格 / 指数增长估值)
```

| 状态 | 得分 | 说明 |
|------|------|------|
| < 0.45 | +2 | 抄底区，价格严重低估 |
| 0.45 - 0.8 | +1.5 | 定投区（激进） |
| 0.8 - 1.2 | +1 | 定投区（稳健） |
| 1.2 - 2.0 | 0 | 等待区 |
| > 2.0 | -1 | 可能高估 |
| > 3.0 | -2 | 泡沫区 |

**数据来源**: coinglass.com, ahr999.com

---

**原理**: 基于对数回归模型的九色带估值

| 色带 | 得分 | 说明 |

| 深蓝（Fire Sale） | +2 | 极端折价，历史最佳买点 |
| 浅蓝（BUY!） | +1.5 | 积累区 |
| 绿色（Accumulate） | +1 | 吸筹区 |

| 黄色（HODL!） | 0 | 公允价值 |
| 浅橙（Is this a bubble?） | -0.5 | FOMO升温 |
| 深橙（FOMO intensifies） | -1 | 泡沫化 |
| 红色（Sell. Seriously, SELL!） | -1.5 | 超买 |
| 深红（Maximum bubble territory） | -2 | 极端泡沫 |

**数据来源**: blockchaincenter.net/bitcoin-rainbow-chart

---

### 1.3 MVRV Z-Score（10%）

**公式**:
```
Z-Score = (市值 - 实现市值) / 市值的标准差
```

| 状态 | 得分 | 说明 |
|------|------|------|
| < 0 | +2 | 严重低估，投降区 |
| 0 - 1 | +1 | 低估 |
| 1 - 2 | +0.5 | 偏低估 |
| 2 - 4 | 0 | 中性区间 |
| 4 - 7 | -1 | 高估 |
| > 7 | -2 | 严重高估，顶部信号 |

**注意**: 该指标呈边际递减趋势，每轮牛市峰值在降低

## 计算示例

假设BTC各因子得分如下：

**基础因子**:

| 因子 | 得分 | 权重 | 加权分 |
|------|------|------|--------|
| AHR999 | +1 | 12% | +0.12 |
| 彩虹图 | +0.5 | 10% | +0.05 |
| MVRV Z-Score | +0.5 | 10% | +0.05 |
| 减半周期 | 0 | 8% | 0 |
| 200MA偏离 | +1 | 8% | +0.08 |
| 梅耶倍数 | 0 | 7% | 0 |
| 交易所余额 | +1 | 8% | +0.08 |
| 稳定币市值 | +0.5 | 6% | +0.03 |
| ETF净流入 | +1 | 8% | +0.08 |
| 恐惧贪婪 | +1.5 | 8% | +0.12 |
| 资金费率 | +1 | 8% | +0.08 |
| NUPL | +1 | 7% | +0.07 |
| **基础合计** | - | 100% | **+0.76** |

**基础胜率**: (0.76 + 2) / 4 × 100% = **69%**

**宏观调整**:
- 美联储暂停加息: +2%
- DXY 103: 0%
- 纳指新高: +2%
- 调整合计: +4%

**最终胜率**: 69% + 4% = **73%** → 可以建仓

此因子体系主要针对 **BTC**，部分指标可用于 **ETH**：

| 因子 | BTC适用 | ETH适用 | 山寨币 |
|------|---------|---------|--------|
| AHR999 | ✅ | ❌ | ❌ |
| 彩虹图 | ✅ | ✅（有ETH版本）| ❌ |

| ETF流入 | ✅ | ✅（ETH ETF）| ❌ |
| 其他 | ✅ | ✅ | 部分 |

### 山寨币评估

山寨币需要额外考虑：
1. 与BTC相关性
2. 项目基本面（TVL、活跃地址、开发活跃度）
3. 代币经济学（解锁进度、通胀率）
4. 交易所上架情况
5. 社区活跃度
