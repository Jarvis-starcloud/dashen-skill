# 基金胜率评估因子

## 因子总览

共8个核心因子，覆盖四大维度：

| 维度 | 因子数量 | 权重合计 |
|------|----------|----------|
| 趋势 | 3个 | 40% |
| 估值 | 2个 | 23% |
| 资金 | 2个 | 22% |
| 情绪 | 1个 | 15% |

---

## 适用范围

本因子体系适用于：
- ✅ 主动管理型股票基金
- ✅ 指数基金（ETF、LOF）
- ✅ 混合型基金
- ⚠️ 债券基金（需调整权重）
- ⚠️ QDII基金（需加宏观因子）
- ❌ 货币基金（无需评估）

---

## 一、趋势维度（40%）

### 1.1 近1年排名分位（15%）

**指标**: 基金近1年收益在同类基金中的排名百分位

| 状态 | 得分 | 说明 |
|------|------|------|
| 前10% | +2 | 顶尖选手 |
| 前10-25% | +1.5 | 优秀 |
| 前25-50% | +0.5 | 中上 |
| 前50-75% | -0.5 | 中下 |
| 后25% | -1.5 | 落后 |
| 后10% | -2 | 垫底 |

**注意**: 
- 结合3年、5年排名交叉验证
- 短期排名波动大，长期排名更稳定

---

### 1.2 最大回撤（12%）

**指标**: 近3年最大回撤幅度

| 状态 | 得分 | 说明 |
|------|------|------|
| < 10% | +2 | 极低回撤，风控优秀 |
| 10-15% | +1.5 | 低回撤 |
| 15-20% | +1 | 正常 |
| 20-30% | 0 | 中等回撤 |
| 30-40% | -1 | 较高回撤 |
| > 40% | -2 | 高回撤，风控差 |

**配合**: 结合最大回撤修复时间

---

### 1.3 夏普比率（13%）

**公式**: 
```
夏普比率 = (基金收益率 - 无风险利率) / 收益率标准差
```

| 状态 | 得分 | 说明 |
|------|------|------|
| > 2.0 | +2 | 卓越的风险调整收益 |
| 1.5 - 2.0 | +1.5 | 优秀 |
| 1.0 - 1.5 | +1 | 良好 |
| 0.5 - 1.0 | 0 | 一般 |
| 0 - 0.5 | -1 | 较差 |
| < 0 | -2 | 负收益或高波动 |

**逻辑**: 每承担1单位风险获得多少超额收益

---

## 二、估值维度（23%）

### 2.1 跟踪指数估值分位（15%）

**指标**: 基金跟踪/重仓的指数当前PE/PB分位

| 状态 | 得分 | 说明 |
|------|------|------|
| < 20%分位 | +2 | 历史极低估 |
| 20-30%分位 | +1.5 | 低估 |
| 30-50%分位 | +0.5 | 偏低估 |
| 50-70%分位 | 0 | 合理 |
| 70-80%分位 | -1 | 偏高估 |
| > 80%分位 | -2 | 历史高估 |

**适用**: 
- 指数基金：直接看跟踪指数
- 主动基金：看重仓股所属行业指数

**数据来源**: 且慢估值、理杏仁、乌龟量化

---

### 2.2 基金经理任期与业绩（8%）

**指标**: 基金经理任职时间 + 任期内业绩

| 状态 | 得分 | 说明 |
|------|------|------|
| 任期 >5年 且 年化>15% | +2 | 久经考验的老将 |
| 任期 3-5年 且 年化>10% | +1 | 稳定优秀 |
| 任期 1-3年 且 业绩尚可 | 0 | 待观察 |
| 任期 <1年 | -1 | 新手或刚换人 |
| 频繁更换经理 | -2 | 管理混乱 |

**注意**: 经理更换后需重新评估

---

## 三、资金维度（22%）

### 3.1 规模变化（10%）

**指标**: 近6个月基金规模变化

| 状态 | 得分 | 说明 |
|------|------|------|
| 稳定增长（10-30%） | +1 | 健康增长 |
| 小幅变动（±10%） | +0.5 | 稳定 |
| 快速增长（>50%） | -1 | 热钱涌入，可能影响操作 |
| 大幅缩水（>30%） | -1.5 | 资金撤离 |
| 规模过小（<1亿） | -1 | 清盘风险 |
| 规模过大（>500亿） | -0.5 | 船大难掉头 |

**逻辑**: 规模是业绩的敌人，但也要避免太小

---

### 3.2 机构持仓占比（12%）

**指标**: 机构投资者持仓占比变化

| 状态 | 得分 | 说明 |
|------|------|------|
| 机构占比 >50% 且增加 | +2 | 机构高度认可 |
| 机构占比 30-50% | +1 | 机构参与 |
| 机构占比 10-30% | 0 | 散户为主 |
| 机构占比 <10% | -0.5 | 纯散户基金 |
| 机构大幅减持 | -1.5 | 机构撤离 |

**数据来源**: 基金季报、年报

---

## 四、情绪维度（15%）

### 4.1 申赎比（15%）

**指标**: 近季度申购份额 / 赎回份额

| 状态 | 得分 | 说明 |
|------|------|------|
| 净申购 >20% | +1 | 资金流入 |
| 净申购 5-20% | +0.5 | 温和流入 |
| 基本持平（±5%） | 0 | 稳定 |
| 净赎回 5-20% | -1 | 资金流出 |
| 净赎回 >20% | -2 | 大量赎回，可能被动卖出 |

**注意**: 大额赎回可能导致基金被动卖出优质持仓

## 特殊基金类型调整

### 指数基金

指数基金主要看估值和跟踪误差，因子权重调整：

| 维度 | 原权重 | 调整后 |
|------|--------|--------|
| 趋势（排名/回撤/夏普） | 40% | 20% |
| 估值（指数分位） | 23% | **50%** |
| 资金（规模/机构） | 22% | 15% |
| 情绪（申赎） | 15% | 15% |

新增因子：
- 跟踪误差（<0.5%为优）
- 费率（越低越好）
- 流动性（日成交额）

### 债券基金

债券基金需要考虑利率环境，因子权重调整：

| 维度 | 原权重 | 调整后 |
|------|--------|--------|
| 趋势 | 40% | 25% |
| 估值 | 23% | 10% |
| 资金 | 22% | 25% |
| 情绪 | 15% | 10% |
| **利率环境** | - | **30%** |

利率环境因子：
- 降息预期 → +2
- 利率平稳 → 0
- 加息预期 → -2

### QDII基金

需要叠加汇率因子：

| 汇率趋势 | 调整 |
|----------|------|
| 人民币贬值预期 | +5%（利好外币资产） |
| 汇率平稳 | 0 |
| 人民币升值预期 | -5%（利空外币资产） |

## 计算示例

假设某主动型股票基金各因子得分：

| 因子 | 得分 | 权重 | 加权分 |
|------|------|------|--------|
| 近1年排名分位 | +1.5 | 15% | +0.225 |
| 最大回撤 | +1 | 12% | +0.12 |
| 夏普比率 | +1 | 13% | +0.13 |
| 指数估值分位 | +1.5 | 15% | +0.225 |
| 经理任期业绩 | +1 | 8% | +0.08 |
| 规模变化 | +0.5 | 10% | +0.05 |
| 机构持仓 | +1 | 12% | +0.12 |
| 申赎比 | +0.5 | 15% | +0.075 |
| **合计** | - | 100% | **+1.025** |

**胜率计算**: (1.025 + 2) / 4 × 100% = **75.6%** → 可以建仓

## 基金筛选流程

### 第一步：排雷

排除以下基金：
- ❌ 规模 < 1亿（清盘风险）
- ❌ 成立 < 1年（数据不足）
- ❌ 近1年排名后30%
- ❌ 最大回撤 > 40%

### 第二步：初筛

保留满足条件的基金：
- ✅ 近1年排名前50%
- ✅ 夏普比率 > 0.5
- ✅ 基金经理任期 > 1年

### 第三步：精选

用本因子体系评分，选择胜率 > 60% 的基金

### 第四步：组合

- 不同风格搭配（成长+价值）
- 不同行业搭配（避免过度集中）
- 不同基金公司搭配
