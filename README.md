# 大神 (DaShen) - 买入胜率预测系统

> 📊 多因子量化评分系统，预测股票/虚拟货币/基金的买入胜率

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/Jarvis-starcloud/dashen-skill)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

## 📖 简介

**大神**是一套基于多因子量化评分的投资胜率预测系统，通过整合趋势、估值、资金、情绪四大维度的多个指标，为股票、虚拟货币、基金的买入决策提供概率化参考。

### 核心理念

> 不预测具体涨跌幅，只回答一个问题：**当前时点买入，赢的概率有多大？**

### 设计原则

1. **多维交叉验证** - 单因子会失效，多维度才稳健
2. **量化减少偏见** - 数字说话，不靠感觉
3. **可回测可优化** - 框架固定，权重可调
4. **承认局限性** - 历史不代表未来，黑天鹅无法预测

## 📊 四维评分体系

所有资产类型共享统一框架：

| 维度 | 权重 | 核心问题 |
|------|------|----------|
| **趋势** | 30% | 当前处于什么周期位置？方向是什么？ |
| **估值** | 25% | 价格相对于内在价值是贵还是便宜？ |
| **资金** | 25% | 聪明钱在进还是在出？ |
| **情绪** | 20% | 市场是恐惧还是贪婪？ |

## 📈 胜率计算

```
单因子得分范围：[-2, +2]
总分 = Σ(因子得分 × 权重)
标准化胜率 = (总分 + 2) / 4 × 100%
```

### 胜率区间解读

| 胜率 | 信号 | 建议 |
|------|------|------|
| **70%+** | ★★★★★ | 强烈买入，可重仓 |
| **55-70%** | ★★★★☆ | 可以建仓，分批买入 |
| **45-55%** | ★★★☆☆ | 观望，等待信号 |
| **30-45%** | ★★☆☆☆ | 谨慎，考虑减仓 |
| **<30%** | ★☆☆☆☆ | 回避，不宜介入 |

## 🚀 使用方法

### 作为 WorkBuddy Skill 使用

1. 将本仓库克隆到 `~/.workbuddy/skills/dashen-skill/`
2. 在对话中使用触发词召唤：
   - "大神帮我看看 XXX"
   - "XXX 现在能买吗？"
   - "预测一下 XXX 的胜率"

### 手动使用

1. 确定资产类型（股票/加密/基金）
2. 参考对应因子文档收集数据
3. 按权重计算各因子得分
4. 得出最终胜率

## ⚠️ 风险提示

1. 本系统仅供参考，**不构成投资建议**
2. 历史规律不代表未来表现
3. 黑天鹅事件（战争、政策突变等）无法预测
4. 建议结合自身风险承受能力决策
5. **任何投资都有亏损风险**
