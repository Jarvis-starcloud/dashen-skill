# 📊 大神 (DaShen) v1.1.0 - 买入胜率预测系统

多因子量化评分系统，预测股票/虚拟货币/基金的买入胜率。

## 功能概览

回答一个核心问题：**当前时点买入，赢的概率有多大？**

### 四维评分体系

| 维度 | 默认权重 | 核心问题 |
|------|----------|----------|
| 趋势 | 30% | 当前处于什么周期位置？方向是什么？ |
| 估值 | 25% | 价格相对于内在价值是贵还是便宜？ |
| 资金 | 25% | 聪明钱在进还是在出？ |
| 情绪 | 20% | 市场是恐惧还是贪婪？ |

### 资产覆盖

- **股票（A股）**：14个核心因子
- **虚拟货币**：12个核心因子 + 3个宏观调整因子（±10%）
- **基金**：8个核心因子
- **宏观环境**：7个宏观因子

### v1.1.0 新增

- 🔄 **动态权重调整机制**：根据宏观环境（牛市/中性/熊市/危机）自动切换四维权重
- 🔗 **Skill 互操作**：接收 financial-shrimp 宏观研判 → 动态选择权重模式
- ⚠️ **分歧处理规则**：定性和定量不一致时的标准化处理流程
- 📊 危机模式权重矩阵（VIX>40 时情绪维度权重提升至 40%）

## 胜率计算

```
总分 = Σ(因子得分 × 动态权重)
标准化胜率 = (总分 + 2) / 4 × 100%
```

| 胜率区间 | 信号强度 | 建议操作 |
|----------|----------|----------|
| 70%+     | ★★★★★   | 强烈买入 |
| 55-70%   | ★★★★☆   | 可建仓   |
| 45-55%   | ★★★☆☆   | 观望     |
| 30-45%   | ★★☆☆☆   | 谨慎     |
| <30%     | ★☆☆☆☆   | 回避     |
